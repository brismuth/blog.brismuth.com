How to send bulk texts with Google voice

2018-08-26 — Written by Brian Smith

Sometimes you want to send a text message to a large group of people. MMS isn’t a great approach — you’d be starting a group chat with a bunch of people that may not even have each other’s contact info. Some people may be annoyed if you do that, especially if others start chatting in the thread. If you ever need to add someone or remove someone from the group your only option is to start a brand new MMS thread, and you can’t stop people from chatting on the old threads. Additionally, MMS support varies from carrier to carrier, so some people may not even receive your message. It quickly becomes a mess.

Alternatively, you can send an individual text message to each person. SMS doesn’t suffer from the problems that MMS suffers from, but manually sending a message to each person can be time consuming. Luckily, you can easily automate the text message sending process with a Google Voice account and a Chrome extension (full disclosure — I built the extension).

The Google Voice Bulk Texter in action

Step One: Sign up for Google Voice

Google Voice is a VOIP service that comes free with your Google account. It basically gives you a phone number that you can place calls and send texts from online (or through Google’s app). You can set up Google Voice here: https://voice.google.com/.

Step Two: Install the Firefox add-on / Chrome Extension

Chrome extensions and Firefox add-ons are small software programs that can customize your browsing experience.

The easiest way to use the Google Voice Bulk Texter is to install it as a Firefox add-on here: https://addons.mozilla.org/en-US/firefox/addon/bulk-texter/.

If you use Chrome, you can install it manually by following these instructions: https://github.com/brismuth/google-voice-bulk-texter#install-it. The extension is open source, and you can view all the program’s source code here: https://github.com/brismuth/google-voice-bulk-texter.

Step Three: Use the Extension

Navigate to https://voice.google.com/ in your Firefox or Chrome browser. You should now see an icon like this to the right of your URL bar:

The Google Voice Bulk Texter Icon

You can now click that icon to see the bulk texting menu:

The bulk texting menu

As shown in the menu, you can use the extension to send messages to a line-separated list of people. You can even pipe in the contact’s name for a personalized touch! I’ve been using the extension regularly since building it to send up to 130 text messages at once to my church group, and it’s worked great.

If you run into any issues or you want to request a new feature, please follow the process here and I’ll respond as soon as possible: https://github.com/brismuth/google-voice-bulk-texter/blob/master/support.md
