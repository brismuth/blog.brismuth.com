Steam In-Home Streaming on a Chromebook

2016-06-16 — Written by Brian Smith

Yes, it is possible. And it works great!

I’ve been doing this for a while and it’s worked great for me, so I thought I’d share my learnings.

Note that this guide will only work on chromebooks with Intel processors.

Step 1: Install Ubuntu using Crouton

First off, you’ll want to install Ubuntu on your chromebook using Crouton. The steps for doing that have already been covered on many blogs, so I won’t go into details there. You should be able to find a guide relatively easily with a quick google search.

This is the command I used to setup my Crouton chroot: sudo sh ~/Downloads/crouton -r trusty -t unity-desktop -t cli-extra -t audio -t keyboard

Step 2: Install Steam

Once you’re in the Crouton environment, you’ll of course want to install Steam. This is very easy, you’ll just download the installer from Steam’s website and double click it after downloading it to Ubuntu.

Step 3: Open the necessary ports on your firewall

ChromeOS by default blocks the ports that Steam uses for in-home streaming. You can change that easily with iptables in your terminal, which you can open with ctrl+alt+t: sudo apt-get install iptables; sudo iptables -A INPUT -p udp --dport 27036 -j ACCEPT; sudo iptables -A INPUT -p udp --dport 37804 -j ACCEPT; sudo iptables -A INPUT -p tcp --dport 37804 -j ACCEPT; sudo iptables -A INPUT -p tcp --dport 27036 -j ACCEPT; sudo iptables -I INPUT -p udp --dport 27031 -j ACCEPT; sudo iptables -I INPUT -p udp --dport 27036 -j ACCEPT; sudo iptables -I INPUT -p tcp --dport 27036 -j ACCEPT; sudo iptables -I INPUT -p tcp --dport 27037 -j ACCEPT;

You will need to run the iptables commands every time you want to use Steam In-Home Streaming, after each reboot. An easy way to do this is to add a bash alias for them: echo "alias steamStream=&#39; sudo iptables -A INPUT -p udp --dport 27036 -j ACCEPT; sudo iptables -A INPUT -p udp --dport 37804 -j ACCEPT; sudo iptables -A INPUT -p tcp --dport 37804 -j ACCEPT; sudo iptables -A INPUT -p tcp --dport 27036 -j ACCEPT; sudo iptables -I INPUT -p udp --dport 27031 -j ACCEPT; sudo iptables -I INPUT -p udp --dport 27036 -j ACCEPT; sudo iptables -I INPUT -p tcp --dport 27036 -j ACCEPT; sudo iptables -I INPUT -p tcp --dport 27037 -j ACCEPT;&#39;" &gt;&gt; ~/.bash_aliases

Now, each time you reboot, you’ll just need to open your terminal, and type steamStream to open the necessary ports for streaming.

Step 4: Start streaming!

Open or restart your Steam client in Crouton, and you will see the option to stream from your other computers on your local network.

Ready to stream on ChromeOS
